# 页面设计文档（桌面优先）

## 全局设计规范

### Layout

- 桌面端基准宽度：1200px 内容容器（居中），两侧留白。
- 栅格：12 列 Grid（间距 24px），表单与卡片使用 8/16/24px 间距体系。
- 响应式：
  - ≥1024px：双列/三列卡片布局；表单左右分组。
  - 768–1023px：单列为主，重要信息置顶。
  - <768px：表单分段折叠（可选），按钮全宽。

### Meta Information（通用）

- 默认 Title 模板：`两人八字适配 - {页面名}`
- 默认 Description：强调“输入出生信息→生成分析/建议/行动计划→可保存回看”。
- Open Graph：`og:title`、`og:description`、`og:type=website`、`og:image`（产品默认封面）。

### Global Styles

- 背景色：#0B1020（深色）或 #F7F8FA（浅色，二选一保持一致）；内容卡片使用对比色。
- 主色（Accent）：#6D5EF3；警示：#F04438；成功：#12B76A。
- 字体：中文优先系统字体；字号阶梯：12/14/16/20/24/32。
- 按钮：主按钮实心、次按钮描边；hover 统一提升亮度与阴影；disabled 降低对比度。
- 链接：主色下划线 hover 出现；避免大段正文蓝色噪点。
- 动效：仅用于状态反馈（生成中 skeleton、按钮 loading），避免花哨。

## 1) 首页（测算入口）

### Meta

- Title：两人八字适配 - 测算入口
- Description：填写双方出生信息，生成八字匹配分析、相处建议与行动计划。

### Page Structure

- 顶部导航（固定高度 64px）
- Hero 介绍区
- 信息录入区（核心）
- 我的报告列表区
- 页脚（免责声明/隐私）

### Sections & Components

1. 顶部导航 Navbar
   - 左侧：Logo + 产品名
   - 右侧：
     - 未登录：登录按钮
     - 已登录：邮箱展示 + 退出
     - 管理员：后台入口
2. Hero
   - 标题：一句话价值主张
   - 副文案：强调“生成三段结果 + 可保存回看”
   - 注意：提供“仅供参考”免责声明提示
3. 信息录入表单（Card）
   - 结构：左右两列（A 与 B），每列字段一致
   - 字段：出生日期（必填）、出生时间（可选）、出生地（可选）、称呼/姓名（可选）、性别（可选）
   - 校验：必填未填时在字段下方提示；日期格式/时间格式校验
   - CTA：主按钮“生成报告”
   - 状态：
     - 未登录点击：弹出提示条（toast）引导登录
     - 提交中：按钮 loading + 表单禁用
4. 我的报告列表（Card + Table/List）
   - 列：创建时间、状态（Tag）、操作（查看）
   - 空状态：引导你先生成第一份报告

### Interaction

- 生成成功：自动跳转报告详情页
- 生成失败：在列表条目上显示失败 Tag，点入详情可重试

## 2) 登录页

### Meta

- Title：两人八字适配 - 登录
- Description：使用邮箱验证码快速登录以保存与回看报告。

### Page Structure

- 居中单卡片布局（最大宽度 420px）

### Sections & Components

1. 登录卡片
   - 邮箱输入
   - 按钮：发送验证码
   - 验证码输入
   - 按钮：验证并登录
   - 错误区：显示发送失败/验证码错误/频率限制
2. 登录态区域（已登录时）
   - 展示当前邮箱
   - 退出按钮

### Interaction

- 发送验证码成功：聚焦验证码输入框 + 倒计时提示
- 登录成功：跳转首页并保持填写内容（若来自首页触发）

## 3) 报告详情页

### Meta

- Title：两人八字适配 - 报告详情
- Description：查看八字适配分析、相处建议与行动计划。

### Page Structure

- 顶部导航
- 内容区：左侧主内容（8 列），右侧信息侧栏（4 列）

### Sections & Components

1. 报告头部 Header
   - 标题：双方称呼（如未填则用“TA/对方”）+ 创建时间
   - 状态：生成中/成功/失败（Tag）
   - 操作：复制全文、失败时“重试生成”
2. 主内容（3 个折叠/锚点区块）
   - 区块 1：八字适配分析（长文排版，段落间距 12–16px）
   - 区块 2：相处建议（要点列表 + 示例话术可用引用样式）
   - 区块 3：行动计划（7 天/30 天分组的任务清单样式）
3. 侧栏 Info Panel
   - 展示输入摘要（出生日期/时间/地）
   - 免责声明与隐私提示

### Interaction

- 生成中：Skeleton 占位 + “预计 10–30 秒”提示
- 失败：显示错误原因（可读文本）+ 重试按钮

## 4) 后台管理页

### Meta

- Title：两人八字适配 - 后台
- Description：管理提示词模板、报告与基础配置。

### Page Structure

- 左侧侧边栏（固定宽度 240px）+ 右侧工作区

### Sections & Components

1. Sidebar
   - 菜单：模板管理 / 报告管理 / 基础配置
2. 模板管理（表格 + 编辑抽屉）
   - 表格列：名称、模型、是否默认、更新时间、操作
   - 编辑抽屉：system prompt、user prompt template、模型名、默认开关
3. 报告管理（表格 + 详情弹层）
   - 筛选：状态、时间范围
   - 详情弹层：输入 JSON（可折叠）+ 输出三段文本 + 失败原因 + 重试
4. 基础配置
   - 默认模板选择
   - 频率限制提示文案、最大输出长度配置（展示为运营参数，不暴露密钥）

### Access & Empty States

- 非管理员访问：显示“无权限”页面状态（同页内提示返回首页）
